# 🚀 Қойнауу ва Істі Бастау

## Бағдарламалау Жабдықтарын Орнату

### 1️⃣ Python Пакеттерін Орнату

Терминалда (cmd, PowerShell немесе VS Code терминалы) төмендегіні жіберіңіз:

```bash
pip install -r requirements.txt
```

Немесе Python 3.9 немесе одан жоғарысы болса:

```bash
py -m pip install -r requirements.txt
```

### 2️⃣ Node.js Пакеттерін Орнату

``bash
npm install
```

### 3️⃣ Іске Қосу

**步骤 1: Flask Серверін Іске Қосу**

Терминалда:
```bash
python main.py
```

Немесе:
```bash
python -m flask run
```

Бәрі дұрыс жүргені білгенде, төмендегі хабарлама шығуы керек:
```
 * Running on http://127.0.0.1:5000
```

**Адым 2: React Қосымшасын Орнату (Жаңа Терминалда)**

```bash
npm start
```

React сервері `http://localhost:3000` адресінде ашылады

## ✨ Жаңа Ерекшеліктер

### 1. SQL Дәлектігі (SQLite)
- **Қашықтай барлығы**: Барлық книгалармен категориялар `books.db` деген SQL дәлектігінде сақталады
- **Резервтік**: Егер Flask құлап төсек, React `localStorage` пайдалана отырып жұмыс істейді
- **Автоматты**: `main.py` іске қосылған кезде таблицалар өздіктен құрылады

### 2. Дизайн Жағдайташ
- 🎨 **Түс схемасы**: Биік деңгейліктік градиент (құмыра, сәнгі, қызған)
- ✨ **Анимациялар**: Сайық, селіл, масштаб, айналу эффектілері
- 🌊 **Өңем сімпатизм**: Барлық батындар мен карталарға плавин бұлшықы
- 📱 **Мобильді**: Барлық құрылғыларда керемет көрінеді

### 3. API Интеграциясы
- Китап қосу, өңдеу, жою Flask API арқылы
- Автоматты база деректерге сақтау
- Fallback механизмі localStorage-ке

## 🔧 Мәслелені шешу

### Flask сервері іске қосылмады?
```
python main.py
```
Қайталап көрсеңіз. Выпуск болсын:
- Python орнатылғаны тиіс
- `pip install` орындалғаны тиіс

### React сервері қосылмады?
```
npm install
npm start
```

### Дәлект коршасын тағайындау
`books.db` файлын жою және `python main.py` қайта іске қосады.

## 📚 API Нүхтендіктері

Flask API төмендегі жайлау нүхтелерін ұсынады:

```
GET  /api/books              # Барлық книгаларды алу
POST /api/books              # Жаңа книга қосу
PUT  /api/books/<id>         # Книгаберді өңдеу
DELETE /api/books/<id>       # Книгабердің жою

GET /api/categories          # Категориялар
POST /api/categories         # Категория қосу
```
