# 📚 Мир Книг - Интернет магазин книг

Полнофункциональное веб-приложение для продажи книг с полной администраторской панелью, построенное на React.

## 🚀 Функциональность

### Для покупателей:
- 📖 Просмотр каталога книг с различными категориями
- 🔍 Поиск книг по названию или автору
- 🛒 Корзина покупок с управлением количеством
- 💳 Оформление заказов
- 📱 Адаптивный дизайн

### Для администраторов:
- 📚 **Управление книгами**: добавление, редактирование, удаление
- 🏷️ **Управление категориями**: создание новых и удаление старых
- 👥 **Управление пользователями**: добавление и удаление пользователей
- 📦 **Управление заказами**: отслеживание и изменение статуса заказов

## 📋 Требования

- Node.js (версия 14 или выше)
- npm или yarn
- Python (версия 3.7 или выше)
- pip (менеджер пакетов Python)

## 🛠️ Установка

### Фронтенд (React)

1. Откройте папку проекта в VS Code
2. Откройте терминал и выполните команды:

```bash
# Установка зависимостей
npm install

# Запуск разработчицкого сервера
npm start
```

Приложение откроется на `http://localhost:3000`

### Бэкенд (Flask с SQLite)

В отдельном терминале:

```bash
# Установка зависимостей Python
pip install -r requirements.txt

# Запуск Flask сервера
python main.py
```

Flask сервер будет работать на `http://localhost:5000`

**Важно!** Убедитесь, что оба сервера (React на :3000 и Flask на :5000) запущены одновременно для полной функциональности базы данных.

## 🎨 Технологии

### Фронтенд:
- **React** - UI библиотека
- **React Router** - Маршрутизация
- **Context API** - Управление состоянием
- **CSS3 с анимациями** - Стилизация и красивые переходы

### Бэкенд:
- **Flask** - Python веб-фреймворк
- **Flask-SQLAlchemy** - ORM для работы с БД
- **Flask-CORS** - Поддержка CORS
- **SQLite** - Легкая встроенная база данных

## 🔐 Доступ к админ панели

Панель администратора доступна по адресу `/admin`

**Функции:**
- 📚 **Книги**: Добавляйте новые книги, редактируйте существующие, удаляйте устаревшие
- 🏷️ **Категории**: Создавайте и удаляйте категории книг
- 👥 **Пользователи**: Управляйте списком пользователей
- 📦 **Заказы**: Отслеживайте заказы и обновляйте их статус

## 📝 Использование

### Добавление новой книги (Админ):
1. Перейдите в `⚙️ Админ` → `📚 Книги`
2. Нажмите `➕ Добавить книгу`
3. Заполните форму с деталями книги
4. Выберите эмодзи для обложки
5. Нажмите `✔️ Добавить`

### Поиск и фильтрация книг:
1. На странице `📚 Книги` используйте поле поиска
2. Выбирайте категорию из фильтров
3. Все книги обновляются в реальном времени

### Оформление заказа:
1. Добавьте книги в корзину
2. Перейдите в `🛒 Корзина`
3. Отрегулируйте количество товаров
4. Нажмите `Оформить заказ`
5. Заказ появится в админ панели

## 💾 Сохранение данных

- **Книги и категории**: Сохраняются в SQLite базе данных (при запущенном Flask сервере)
- **Корзина и заказы**: Сохраняются в `localStorage` браузера
- **Резервное хранилище**: Если Flask сервер недоступен, приложение использует `localStorage` как резервный вариант
