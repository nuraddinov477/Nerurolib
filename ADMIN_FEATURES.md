# 🎛️ Admin Panel - To'liq Imkoniyatlar

## 📚 Yangi Xususiyatlar

### 1. 👥 Kengaytirilgan Foydalanuvchilar Boshqaruvi

#### **Real vaqt rejimida kuzatish**
- 🟢 **Onlayn foydalanuvchilar** - kim hozir saytda ekanligini ko'ring
- ⏱️ **Vaqt hisobi** - har bir foydalanuvchi qancha vaqt saytda bo'lganini ko'ring
- 📱 **Qurilma ma'lumotlari** - mobile, tablet yoki desktop

#### **Foydalanuvchilarni boshqarish**
- ✅ **Qo'shish** - yangi foydalanuvchilar yarating
- 🗑️ **O'chirish** - foydalanuvchilarni tizimdan olib tashlang
- 🔒 **Bloklash** - keraksiz foydalanuvchilarni bloklang
- 🔓 **Blokdan chiqarish** - bloklangan foydalanuvchilarni qayta faollashtiring

#### **Batafsil statistika**
Har bir foydalanuvchi uchun:
- 📊 **Umumiy vaqt** - saytda o'tkazgan umumiy vaqt
- 📖 **O'qilgan kitoblar** - qaysi kitoblarni o'qigani
- 📥 **Yuklab olingan kitoblar** - qaysi kitoblarni yuklab olgani
- 🌐 **Sessiyalar tarixi** - qachon va qaysi qurilmadan kirgani
- 🕐 **Oxirgi tashrif** - eng so'nggi kirgan vaqti

### 2. 🤖 AI Kitob Generatori

#### **Sun'iy intellekt bilan kitoblar yaratish**
- ✨ **Avtomatik yaratish** - AI yordamida kitoblar generatsiya qiling
- 🎯 **Moslashuvchan sozlamalar**:
  - Kategoriya tanlash
  - Kitoblar soni (1-20)
  - Til tanlash (O'zbek, Rus, Ingliz)
  - Tavsif va muqova qo'shish

#### **Tezkor topshiriqlar**
8 ta tayyor shablonlar:
- 📚 O'zbek klassik adabiyoti
- 💪 Motivatsion kitoblar
- 👶 Bolalar uchun hikoyalar
- 💻 IT va texnologiya
- 📜 Tarixiy romanlar
- 💼 Biznes va muvaffaqiyat
- 🧠 Psixologiya
- 🚀 Fan-fantastika

#### **Kitoblarni ko'rib chiqish**
- 👁️ Yaratilgan kitoblarni ko'rib chiqing
- ✅ Kutubxonaga qo'shing yoki o'chiring
- 📦 Hammasini birlashtirib qo'shing

## ⚙️ Texnik Ma'lumotlar

### **Backend API Endpoints:**
```
GET  /api/analytics/online-users     - Onlayn foydalanuvchilar
GET  /api/analytics/user/:id         - Foydalanuvchi statistikasi
POST /api/admin/users/:id/block      - Foydalanuvchini bloklash
POST /api/ai/generate-books          - AI kitob generatsiyasi
GET  /api/analytics/dashboard        - Dashboard ma'lumotlari
```
